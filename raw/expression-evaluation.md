Expression Evaluation
===

## Problem

Given an expression string array, return the final result of this expression

 Notice
The expression contains only integer, +, -, *, /, (, ). 



## Example

For the expression 2*6-(23+7)/(1+2),
input is
[
  "2", "*", "6", "-", "(",
  "23", "+", "7", ")", "/",
  (", "1", "+", "2", ")"
],

return 2
