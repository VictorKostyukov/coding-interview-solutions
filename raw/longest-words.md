Longest Words
===


Problem
-------

Given a dictionary, find all of the longest words in the dictionary.

Example
-------

Given

    {
      "dog",
      "google",
      "facebook",
      "internationalization",
      "blabla"
    }

the longest words are(is) ["internationalization"].

Given

    {
      "like",
      "love",
      "hate",
      "yes"
    }

the longest words are ["like", "love", "hate"].
