# Permutations II

LintCode 16: http://www.lintcode.com/en/problem/permutations-ii/

## Problem

Given a list of numbers with duplicate number in it. Find all unique permutations.

## Example

For numbers [1,2,2] the unique permutations are:

```
[
  [1,2,2],
  [2,1,2],
  [2,2,1]
]
```
